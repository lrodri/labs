%%%--- Colocar objetivos acá ----%%%%
\objetivos{

\begin{itemize}
\item  Familiarizarse con la instalación de la estación base
\item  Familiarizarse con el sistema operativo airOS
\item  Establecer una contexión punto a punto entre los dos PC
\end{itemize}
}

%%%----- Listar los materiales acá ------%%%
\materiales{

\begin{enumerate}
  \item 2 estaciones base Rocket M5
  \item 4 cables UTP
  \item 2 PC
\end{enumerate}
}


%%% ------- colocar planteamiento y procedimiento
\planteamiento{
\begin{enumerate}

%% Instalación estación base
\item Conecte la estación base al computador tal como indica el manual

%% instalación
\item Enciendalo y verifique que halla conexión tal vez haciendo {\tt ping} a
{\tt 192.168.0.20}. Para ello debe configurar una IP estática a la tarjeta
Ethernet del PC que pertenezca a la misma red.

%% airOS
\item Con un navegador ingrese a esta IP e identifíquese: {\tt user: ubnt} y
{\tt password: ubnt}. Explore el airOS navegando por las diferentes pestañas.

%% airOS

\item En la tarjeta {\sl Network} del airOS, cambie las direcciones IP de las
dos estaciones base a una red diferente de la que trae por omisión la estación
base. Para que se pueda comunicar debe cambiar también la dirección IP del PC.
Antes de salir de cada tarjeta no olvide guardar, presionando {\em change} y luego ...

%% airOS
\item En la tarjeta {\sl Wireless} cambie el {\tt ssid} en las dos estaciones.
Configure una en modo {\em Access Point} y el otro como {\em Station}. No le
coloque ninguna seguridad. Una vez establecida la conexión (vea la intensidad
de la señal en los leds de la estación base) haga {\tt ping} desde {\tt Tools}
a la estación base vecina.

%% conexión
\item Compruebe la conexión entre los dos PC (cada uno conectado a una estación
base) mediante {\tt ping}

%%i airOS
\item Una vez culminadas las pruebas debe realizar un {\em reset} por software
esto se lleva a cabo en la tarjeta de {\em System} en {\em Reset to Factory
Defaults}

\end{enumerate}
}

%%% --- colocar acá las recomendaciones ---%%%
\recomendaciones{

\begin{itemize}

\item Asegurarse de hacerle el {\em reset} por software al equipo, cuando
culminen la práctica, para que se pueda utilizar sin inconvenientes los
estudiantes que continúen

\item Los equipos se entregan y tienen 24 horas para realizar la
práctica, una vez culminado su tiempo deben pasarlos al siguiente grupo
en la lista


\item No utilizar el botón de {\em reset} del equipo.

\item La red con que trabajarán sus equipos y el nombre del {\tt ssid} debe conservarlos todas las prácticas.

\end{itemize}
}
